# 高考志愿填报系统 - 安装使用说明

## 系统简介

高考志愿填报系统是一款基于Web的高校招生数据分析与志愿填报辅助系统，提供以下核心功能：

- 📊 **数据看板**：直观展示招生数据统计信息
- 🔍 **院校分析**：查询各高校的详细录取信息、保研率、学科评估等
- 📚 **专业分析**：了解各专业的录取情况和院校分布
- 🎯 **智能搜索**：按分数、位次、地区、院校标签等条件综合搜索
- 📈 **历史分析**：查看学校和专业的三年录取趋势
- 🔮 **位次预测**：基于历史数据预测2026年录取位次
- 📝 **志愿填报**：智能生成志愿方案，支持Excel/PDF导出

## 安装步骤

### 方式一：便携版（推荐，无需安装）

1. **解压文件**
   - 将下载的压缩包解压到任意目录
   - 例如：`C:\高考志愿填报系统\`

2. **启动系统**
   - 双击运行 `启动系统.bat` 文件
   - 系统会自动启动，并在浏览器中打开访问地址

3. **访问系统**
   - 启动成功后，浏览器会自动打开 `http://localhost:5000`
   - 如果没有自动打开，手动在浏览器中访问上述地址

### 方式二：Python环境安装

如果您的电脑已安装Python 3.9+，可以使用此方式：

1. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

2. **启动系统**
   ```bash
   python app_v2.py
   ```

3. **访问系统**
   - 在浏览器中访问 `http://localhost:5000`

### 方式三：使用便携版Python

如果您需要在没有Python环境的电脑上使用，可以：

1. **下载便携版Python**
   - 访问 https://www.python.org/downloads/
   - 下载 Windows embeddable package (64-bit)
   - 解压到 `python` 文件夹

2. **安装依赖**
   ```bash
   python\python.exe -m pip install -r requirements.txt
   ```

3. **启动系统**
   ```bash
   python\python.exe app_v2.py
   ```

## 使用指南

### 1. 数据看板
访问系统首页，查看整体数据统计和图表分析。

### 2. 院校分析
- 在搜索框中输入学校名称进行搜索
- 点击学校名称查看详细信息，包括：
  - 历年录取分数和位次
  - 院校基本信息（所在城市、办学层次等）
  - 保研率和排名
  - 学科评估结果
  - 专业列表

### 3. 专业分析
- 搜索专业名称查看详细信息
- 查看该专业在各院校的录取情况
- 了解专业的历年分数和位次分布

### 4. 综合搜索
- 支持按以下条件筛选：
  - 分数范围（最低分-最高分）
  - 位次范围（最小位次-最大位次）
  - 地区/城市
  - 院校类型（985、211、双一流等）
- 点击搜索按钮查看结果
- 点击学校名称查看详情

### 5. 历史分析
- **学校排名趋势**：查看热门学校三年的录取位次变化
- **学校列表**：浏览所有有三年数据的学校
- **专业列表**：浏览所有专业
- **历史对比**：选择多个学校进行横向对比
- **学校详情**：查看单个学校的三年详细数据
- **专业详情**：查看单个专业的三年详细数据

### 6. 位次预测
- 查看批量预测的学校排名
- 点击学校查看预测详情和置信区间
- 了解预测算法的依据

### 7. 志愿填报（核心功能）

#### 7.1 创建新方案
1. 点击"新建方案"按钮
2. 输入考生信息：
   - 高考分数
   - 排名位次
   - 科目类型（文科/理科）
3. 选择推荐算法：
   - **均衡型**：综合多因素评分，推荐平衡的志愿
   - **保守型**：侧重稳定性，推荐录取概率高的院校
   - **激进型**：冲一冲更好的学校，但风险略高
4. 设置筛选条件：
   - 地区偏好（可多选）
   - 专业偏好（可多选）
   - 院校类型（985、211、双一流等）
5. 点击"生成志愿"按钮

#### 7.2 调整志愿
- 查看生成的志愿列表
- 根据录取概率和风险等级调整
- 可以拖动志愿进行排序
- 点击删除按钮移除不想要的志愿

#### 7.3 导出方案
- 支持**Excel格式**导出
- 支持**PDF格式**导出
- 导出文件保存在 `exports` 文件夹

#### 7.4 管理多个方案
- 可以创建多个不同的方案进行对比
- 切换查看不同方案
- 删除不需要的方案

## 常见问题

### Q1: 启动时提示端口被占用怎么办？
**A:** 系统默认使用5000端口。如果端口被占用：
1. 关闭占用5000端口的程序
2. 或者在 `app_v2.py` 中修改端口号（找到 `port = 5000` 这一行，修改为其他端口号）

### Q2: 数据加载失败怎么办？
**A:** 请检查以下项目：
1. 确保 `data` 文件夹存在且包含完整的数据文件
2. 确保数据文件格式正确
3. 查看控制台错误信息，根据提示排查问题

### Q3: 浏览器无法打开系统怎么办？
**A:** 尝试以下方法：
1. 检查系统是否成功启动（查看命令行窗口）
2. 手动在浏览器中输入 `http://localhost:5000`
3. 尝试使用不同的浏览器（Chrome、Firefox、Edge等）
4. 检查防火墙是否阻止了5000端口

### Q4: 导出Excel或PDF失败怎么办？
**A:** 检查以下事项：
1. 确保 `exports` 文件夹有写入权限
2. 确保志愿列表不为空
3. 查看控制台错误信息

### Q5: 志愿推荐结果不准确怎么办？
**A:** 志愿推荐基于以下数据：
- 2023-2025年三年的录取数据
- 您的分数和位次
- 选择的筛选条件

如果推荐结果不理想：
1. 尝试调整筛选条件（放宽地区或专业限制）
2. 尝试不同的推荐算法
3. 检查输入的分数和位次是否准确

### Q6: 系统占用内存过高怎么办？
**A:** 系统会缓存数据以提高性能：
1. 重启系统可以释放内存
2. 如果内存不足，可以关闭其他程序
3. 定期清理 `cache` 文件夹中的缓存文件
